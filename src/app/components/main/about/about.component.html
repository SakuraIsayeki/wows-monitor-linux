About wows-monitor

Version: {{version}}

This app was created with Electron and Angular.
It functions as a user interface for data provided by a centralized API written in .NET Core.

This app uses publicly available data from the Wargaming API. By using this app you agree to:

  • fetching data from the API on your behalf
  • processing of fetched data
  • display of fetched data in this app

It only accesses publicly available information (e.g. number of battles, statistics). It does not collect any private data (e.g. email address, number of credits). None of your personal stats are stored on the server.

The color values for the Personal Rating and other stats are provided by wows-numbers.com

If you are interested in the source code for this app check out the repository.
